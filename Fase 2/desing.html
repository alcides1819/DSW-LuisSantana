Sitio web para el aprendizaje de las etiquetas semànticas, listas y enlaces

Este sitio tiene como proposito fomentar el aprendizaje de las tematicas correspondientes a las etiquetas semànticas,
listas y enlaces para asi poder diseñar sitios web utilizando lenguaje HTML.

Etiquetas Semànticas

Semántica se refiere a todo aquello que tiene que ver con el significado de una palabra u expresión. El correcto manejo
semántico de una palabra se refiere a que debe existir una relación entre el uso que se le da a esta palabra, con el significado
de ella misma.
¿Porqué es importante el buen uso de las etiquetas semánticas dentro de HTML?
Podemos resumir su importancia en los siguientes puntos.
Permite una mejor legibilidad del contenido de un documento HTML, tanto para el desarrollador, como para un indexador de contenido.
Al mejorar la legibilidad para los motores de búsqueda mejorará su posicionamiento.
Permite el desarrollo de contenido escalable.
Contenido ordenado y estructurado.
