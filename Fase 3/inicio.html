PROYECTO DE SITIO WEB PERIODO 2021 16-01

INTRODUCCION

Esta activdad se va a desarrollar el sitio web utilizando tècnicas de codificacion HTML y aplicando estilos CSS. Por otro lado se van a tomar las recomendaciones dadas en la guia para llevar a cabalidad lo planteado.

PROPOSITO

Este sitio tiene como proposito fomentar el aprendizaje de las tematicas correspondientes a las etiquetas semànticas, listas y enlaces para asi poder diseñar sitios web utilizando lenguaje HTML.
